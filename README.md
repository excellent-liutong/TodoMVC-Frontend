# 待办事项清单 V2.0
1. 主要功能：
   1. 用户可注册、登录账号，修改账号密码
   2. 用户可增加、编辑、删除、标记完成待办事项，一键删除已完成事项
   3. 用户再次登录可同步已有数据
2. 主要技术栈：
   1. 项目使用`React.js`框架搭建前端
   2. 使用基于`Node.js`的`Express`框架搭建服务端
   3. 使用`MySQL`存储数据库
   4. 使用`Sequelize`作为ORM操作数据库
   5. 使用`JWT`作为登录令牌
   6. 使用`PM2`运维服务端
